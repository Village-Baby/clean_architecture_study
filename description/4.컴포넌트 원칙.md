## 컴포넌트
```
컴포넌트 : 시스템의 구성 요소로 배포할 수 있는 가장 작은 단위다.

컴포넌트가 마지막에 어떤 형태로 배포되든, 잘 설계된 컴포넌트라면 반드시 독립적으로 배포 가능하고 독립적으로 개발 가능한 능력을 갖춰야한다.
```
